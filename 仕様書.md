# アプリ仕様書：Amazon安心検索 (Clean Shopping for Amazon)

## 1. 概要・コンセプト
Amazonの一般商品検索において、「怪しい中国メーカー・出品者」を排除し、信頼できる「メーカー希望小売価格（参考価格）」が表示されやすい「Amazon.co.jp（直販）」の商品のみを素早く検索するためのWebツール。

## 2. ターゲットユーザー
- Amazonでサクラレビューや低品質な類似品に騙されたくない人
- メーカー品や正規販売品を、適正価格（希望小売価格）と比較して買いたい人
- 外出先（スマホ）から手軽に「ハズレのない検索」をしたい人

## 3. 主要機能一覧
### ① 検索キーワード入力機能
- ユーザーが探したい商品名を入力するテキストボックス。

### ② 安心検索（Amazon連携）ボタン
- 入力されたキーワードに対し、以下の検索パラメータを自動付与してAmazonの検索結果画面を別タブで開く。
    - **`&emi=AN1VRQENFRJN5`**: 販売元を「Amazon.co.jp」のみに限定。これにより、マーケットプレイスの怪しい出品者を一括除外。
    - **期待される効果**: 直販商品に絞ることで、メーカー希望小売価格（参考価格）が表示される確率を最大化する。

### ③ サクラ度チェック（サクラチェッカー連携）ボタン
- 同じキーワードを使って、[サクラチェッカー](https://sakura-checker.jp/)の検索結果画面を別タブで開く。
- Amazonの結果と見比べることで、さらに信頼性を担保する。

### ④ Google検索（公式サイト価格）ボタン
- 「入力されたキーワード + 公式サイト + 価格」というクエリでGoogle検索結果を別タブで開く。
- Amazonで気になった商品の「メーカー名＋商品名」を検索欄に入れ直してこのボタンを押すことで、適正価格（定価）を素早くGoogleで確認可能にする。

## 4. 画面構成（1画面）
シンプルさを重視したシングルページ構成。

- **ヘッダー**: アプリ名「Amazon安心検索」
- **メインエリア**: 
    - 検索キーワード入力欄（プレースホルダー：例「モバイルバッテリー」）
    - [ Amazon(直販) ] ボタン（目立つ色：オレンジ系）
    - [ サクラチェッカー ] ボタン（サブ：青色系）
    - [ Googleで価格検索 ] ボタン（サブ：赤色系）
- **フッター**: 使い方のヒント（「Amazon直販のみに絞り込みます」等の注釈）

## 5. 推奨される使用技術
- **言語**: Python
- **フレームワーク**: Streamlit (爆速でWebUIを作成でき、スマホ表示にも対応)
- **公開環境**: Streamlit Community Cloud (GitHub連携により無料でサーバー公開可能)

## 6. 特記事項（技術メモ）

- サクラチェッカー URL生成例:
  `https://sakura-checker.jp/search/[キーワード]/`
